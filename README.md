# Image Classification with CIFAR-10

## Project Goal
This project aims to complete a basic image classification task using a public dataset and a pretrained model.

## Weekly Tasks
- Select a simple public dataset
- Fine-tune a classifier using a pretrained model
- Output training results
- Plot confusion matrix
- Conduct error analysis

## Dataset
CIFAR-10 dataset

## Model
Pretrained ResNet18

## Tools
- Python
- PyTorch
- Google Colab
- GitHub

## Expected Outputs
- Training/validation accuracy and loss
- Confusion matrix
- Misclassified examples
- Brief error analysis
